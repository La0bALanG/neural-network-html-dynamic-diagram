# Quality Checklist

Before final delivery, check:

- The HTML file opens without a build step.
- Inline JavaScript passes a syntax check.
- The title and heading name the algorithm.
- All phases can be reached by play/pause or next/previous controls.
- The current phase is visible in the diagram and in the formula panel.
- Arrows/data flow are directionally clear.
- Tensor/layer shapes and hyperparameters match the user's source or cited references.
- Reference-image structure is reflected when the user provided an image.
- Text is not clipped, overlapped, or compressed into a narrow vertical column.
- SVG/canvas/3D scene is nonblank.
- Mobile layout degrades gracefully with horizontal scrolling for wide technical diagrams.
- The final response links to the HTML with an absolute workspace path.

When exact source details were researched online, cite the sources in the final response.
